# Weather-Timeseries-Data-in-Bangladesh

## Content
The dataset covers over 1200 restaurant information in California. This dataset has 18 fields. The fields are given below:
- Yelp URL
- Restaurant Name
- Street Address
- Zip Code
- City
- State
- Price Range
- Phone
- Rating
- Number of Reviews
- Website
- Menu Link
- Image 1
- Image 2
- Image 3
- Category 1
- Category 2
- Category 3
